## Basic Usage


Available Types:
* `green-quote`
* `yellow-quote`

`green-quote` example:
```html
<blockquote class="bigquote green-quote">
    <p class="quote">Pullquote or callout pullquote or callout</p>
    <p class="byline">byline</p>
</blockquote>
```

`yellow-quote` example:
```html
<blockquote class="bigquote yellow-quote">
    <p class="quote">Pullquote or callout pullquote or callout</p>
    <p class="byline">byline</p>
</blockquote>
```
