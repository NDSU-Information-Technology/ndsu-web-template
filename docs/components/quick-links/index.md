## Basic Usage

```html
<section class="quick-links " role="navigation">
    <h5>Quick links or section of lots of links</h5>
    <ul class="quick-links-3">
        <li><a href="link1">Link 1</a></li>
        <li><a href="link1">Link 2</a></li>
        <li><a href="link1">Link 3</a></li>
        <li><a href="link1">Link 4</a></li>
        <li><a href="link1" target="_blank">Link 1</a></li>
        <li><a href="link1">Link 2</a></li>
        <li><a href="link1" target="_blank">Link 3</a></li>
        <li><a href="link1">Link 4</a></li>
        <li><a href="link1" target="_blank">Link 1</a></li>
        <li><a href="link1">Link 2</a></li>
        <li><a href="link1">Link 3</a></li>
        <li><a href="link1" target="_blank">Link 4</a></li>
    </ul>
</section>
```
