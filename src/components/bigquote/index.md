## Basic Usage


Available Types:
* `green-quote`
* `yellow-quote`

`green-quote` example:
```html
{{> _bigquote text="Pullquote or callout pullquote or callout" source="byline"}}
```

`yellow-quote` example:
```html
{{> _bigquote color="yellow" text="Pullquote or callout pullquote or callout" source="byline"}}
```
